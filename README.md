# Daily Diet

Todo mundo já tentou fazer dieta pelo menos uma vez. E todo mundo sabe como é difícil manter o controle sem perder a cabeça com planilhas ou anotações no papel. O Daily Diet nasceu exatamente pra resolver isso — de forma simples, visual e direto ao ponto.

Com ele você registra o que comeu, marca se estava dentro ou fora da dieta, e acompanha seu progresso ao longo do tempo. Sem julgamentos, só dados.

---

## O que você pode fazer

- Registrar refeições com nome, descrição, data e horário
- Marcar se a refeição estava dentro ou fora da dieta
- Editar ou excluir refeições quando precisar
- Ver seu percentual de refeições saudáveis
- Acompanhar sua sequência atual e sua melhor sequência
- Tudo salvo direto no seu celular, sem precisar de cadastro ou internet

---

## Como rodar o projeto

Você vai precisar ter o **Node.js** instalado e o app **Expo Go** no seu celular.

```bash
# Clone o repositório
git clone https://github.com/Gabrieel13/daily-diet.git

# Entre na pasta
cd daily-diet

# Instale as dependências
npm install

# Inicie o projeto
npx expo start
```

Depois é só escanear o QR code com o **Expo Go** e o app abre direto no seu celular.

---

## Tecnologias utilizadas

- **React Native** com Expo
- **TypeScript**
- **React Navigation** para a navegação entre telas
- **AsyncStorage** para salvar os dados localmente
- **Fonte Nunito Sans** via Google Fonts
